CS 591: Lua Implementation of Assignments

Authors: Steven Jones and Azeeza Eagal

This is a collection of homework assignments from CS 591: Sinless Software Engineering. The overall goal of this collection is to learn how to build a succint and context-aware data miner. More description later


Homework 2: Create a CSV Reader

For this assignment, we created a csv reader that would read each line in a CSV file convert the rows to a list of lists, ignore comments, whitespace, and break each line of a comma. The reader checks for bad lines and wrong number of cells. The runtime for the POM3a datafile was 0.46 seconds.

Homework 3: Create a Sample Class

For this assignment, we created a Sample class. We also created several other classes --- Num, Klass, Goal, Num, Skip, and Sym --- to handle the different attributes given to the column headers. We also created a zitler continous domination predicate function to sort of all the columns given the goal attribute.

Homework 4: Investigate "Peeking"
